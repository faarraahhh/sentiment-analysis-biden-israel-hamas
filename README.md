# sentiment-analysis-biden-israel-hamas
Proyek ini menganalisis sentimen komentar netizen terhadap pernyataan Joe Biden mengenai konflik Israel–Hamas menggunakan data komentar YouTube dari kanal ABC News. Analisis dilakukan untuk mengidentifikasi kecenderungan opini publik terhadap dukungan Amerika Serikat kepada Israel pasca eskalasi konflik pada Oktober 2023.
